# Personal Landing Page

## Project Overview

This project is a responsive personal landing page built using HTMl, CSS, JavaScript.

The goal is to create a clean, fast and professional portfolio website.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)

## Project Constraints

- No frameworks
- No external UI libraries
- Mobile-first responsive design
- Maintain semantic HTML structure
- Avoid unecessary animations
- Keep JavaScript modular and beginner-friendly

## Coding Standards

- Use reusable CSS classes
- Use meaningful file and function names
- Keep code clean and readable
- Separate concerns properly
- Avoid inline styles

## Accessibility Requirements

- Use semantic headings
- Maintain keyboard accessibility
- Proper contrast ratios
- Add alt text for images

## AI Usage Rules

- Generate only task-specific code
- Explain code before generating
- Avoid unnecessary complexity
- Keep solutions maintainable
- Prefer readability over cleverness

## Folder Structure Rules

- styles/ for CSS
- scripts/ for JS
- assets/ for images/icons
- data-driven rendering for projects and skills

## Git Workflow

- Make small meaningful commits
- Commit after completing logical milestones
